## 📌 About The Project
This project is an automated, real-time data analytics and prediction dashboard for the Counter-Strike 2 (CS2) community market. Built with a focus on **Data Engineering** and **Machine Learning**, it aims to help users track their asset portfolios and predict short-term market trends using historical data.

**Live Demo:**[ [Insert your Streamlit Cloud Link Here]](https://cs2-market-ai-forecast.streamlit.app/)

### ✨ Key Features
* **🕷️ Real-time Data Scraper:** A built-in stealth scraper that fetches the latest case prices directly from the Steam Community Market, bypassing standard API rate limits.
* **📈 AI Price Prediction:** Utilizes **Polynomial Regression (Degree 3)** from `scikit-learn` to model market fluctuations and forecast prices for the next 7 days.
* **💼 Portfolio Management:** Interactive UI to manage personal inventory (e.g., holding Fracture Cases) with real-time ROI (Return on Investment) calculations.
* **📊 Advanced Visualizations:** Features interactive Candlestick charts, Simple Moving Averages (SMA 7), and Asset Allocation Donut Charts powered by `Plotly`.
